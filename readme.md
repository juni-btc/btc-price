## 👻 소개

- [x] 비트코인 거래소 시장가 시세 제공(KRW/USD)
- [x] 한국 프리미엄, 환율(USD/KRW), Fear & Greed Index 지수 제공
- [x] BTC 계산기
  - BTC/KRW  <=> KRW/BTC
  - BTC/USD  <=> USD/BTC
  - BTC/SAT
