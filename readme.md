## ⛏️ 기능

<br>

- [x] 비트코인 시장가 제공(USD, KRW)
- [x] 환율(USD/KRW), 김치 프리미엄, 크립토 공포&탐욕 지수 제공
- [x] BTC/KRW, KRW/BTC, BTC/SAT, KRW/SAT 계산기

<br>

## 🏃🏻‍♂️ Getting Started!

### Package Install

```
yarn
or
npm i
```

### Run Dev Server

```
yarn dev
or
npm run dev
```

### Build

```
yarn build
or
npm run build
```
